
  The Outbridge Winamp Plugin can be very helpful in
  understanding Winamp's output plugin architecture.

  It is plugged between Winamp and a "normal" output
  plugin and visualizes which functions are called
  for this plugin in realtime. Parameter and return
  values are shown, too.


  Written by Sebastian Pipping <webmaster@hartwork.org>
